# Algorithm Visualiser

An interactive web application for learning computer science algorithms through step-by-step animations and visualizations.

## Features

- **Searching Algorithms**: Binary Search (Array-based) and Binary Search Tree (Node-based)
- **Sorting Algorithms**: Selection Sort, Insertion Sort, and Quick Sort 
- **Recursion Visualizations**: Conceptual entry points for recursive algorithms 
- **Interactive Controls**: Play, pause, step-through, and speed adjustment for algorithm execution
- **Visual Feedback**: Color-coded elements showing current operations, comparisons, and sorted states

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ``` 

3. Start the development server:
   ```bash
   npm run dev
   ``` 

## Usage

1. Open your browser and navigate to the development server (typically `http://localhost:5173`)
2. Select an algorithm category from the home page
3. Choose a specific algorithm to visualize
4. Use the control panel to:
   - Generate random data or input custom values
   - Play/pause the animation
   - Step through the algorithm execution
   - Adjust animation speed

## Build for Production

To create a production build:

```bash
npm run build
```

This will compile the application into the `dist/` folder.
